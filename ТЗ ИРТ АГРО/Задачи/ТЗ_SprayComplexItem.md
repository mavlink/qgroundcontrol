# Цель
Доработать существующий модуль создания полетного задания для распыления. Добавить необходимые параметры и изменить алгоритм построения гонов для учета препятствий на поле
# Текущий функционал
Файлы модуля (custom/src/SprayPattern/):
1. SprayComplexItem.h 
2. SprayComplexItem.cc 
3. SprayItemEditor.qml - Редактор параметров (в окне FlightPlan справа)
4. SprayMapVisuals.qml - Отображение на карте
5. Spray.SettingsGroup.json

Сейчас миссия создается так же как в SurveyComplexItem но в начале и конце гонов добавляются команды MAV_CMD_DO_SPRAYER. можно указать скорость пролета, высоту и ширину распыления. Для обводки поля используется QGCMapPolygon.
# Требуемый функционал
### Добавить необходимые параметры
1. Угол сетки - угол наклона гонов
2. Стартовая точка - с какого угла поля начинать миссию
3. Норма внесения - вводится в Л/га, далее с учетом высоты полета и ширины распыления вычисляется [SPRAY_PUMP_RATE](https://ardupilot.org/copter/docs/parameters.html#spray-pump-rate) и вносится в полетник
Первые два параметра реализованы в src/MissionManager/SurveyComplexItem. нужно скопировать
### Учет препятствий
Нужно чтобы была возможность добавлять полигоны в которые нельзя залетать. И соответственно миссия перестраивается с учетом препятствий. Полагаю лучший вариант решения это делать как в dji, есть выбор точки(край поля, неполетная зона, препятствие) и кнопка добавить, при нажатии на нее точка добавляется в центр экрана.
